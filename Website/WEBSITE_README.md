# FindAssure Website

This folder contains the public project website for FindAssure, built with React, Vite, and Tailwind CSS.

## Live Website

[https://findassure.vercel.app/](https://findassure.vercel.app/)

## Purpose

The website presents the FindAssure research project for academic reviewers, supervisors, examiners, and visitors. It includes the project overview, research domain, milestones, documents, presentation links, team details, and contact information.

## Tech Stack

- React
- Vite
- Tailwind CSS
- EmailJS for the contact form

## Local Setup

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Run lint checks:

```bash
npm run lint
```

## Content Updates

Most website content is managed from:

```text
src/data/siteContent.js
```

Use this file to update:

- Navigation labels
- Project overview text
- Research domain content
- Milestone statuses
- Document and presentation download links
- Team member details
- Contact information
