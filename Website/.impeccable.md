# Design Context

- Target audience: Academic reviewers, supervisors, examiners, and lecturers evaluating the final-year project.
- Primary use cases:
  - Understand the research problem and solution framing quickly.
  - Review the methodology, milestones, deliverables, and team structure with confidence.
  - Assess the project as a credible institutional AI system rather than a generic showcase site.
- Brand personality / tone: Editorial-tech. Premium, confident, research-driven, and visually distinctive without feeling flashy or consumer-startup-like.
- Performance budget: Balanced motion. Rich polish is welcome, but mobile performance, accessibility, and readability remain hard constraints.
